# Running locally
This code pattern can be run on a local development server with just 3 steps:

1. Clone the repo:
    ```
    git clone https://github.ibm.com/watson-developer-cloud/nlu-pattern.git
    ```

1. Navigate to the cloned repo and install dependencies:
    ```
    npm install
    ```

1. Run the project!
    ```
    npm run dev
    ```
